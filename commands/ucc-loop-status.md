# Loop Status

检查当前验证循环的状态。

## 命令功能

- 若无验证循环在运行，返回提示
- 若正在运行，展示最近一次检查结果
- 汇总失败项与下一步建议

## 执行步骤

1. 读取最近一次 `ucc-loop-start` 记录的检查结果（时间戳 + 各项状态）
2. 若无记录：提示「尚未运行 ucc-loop-start，无历史结果」
3. 若有记录：输出简表：
   - 检查时间
   - 各检查项状态（通过 ✓ / 失败 ✗ / 跳过 —）
4. 如有未解决问题：提示运行 `ucc-loop-start` 重新检查
